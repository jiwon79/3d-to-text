# 3d-to-text
3D 물체인 도넛을 특수문자를 사용해서 보여줍니다.

## Tech Stack

## 주요기능
### 1. 도넛의 크기 조절
왼쪽 상단의 slider를 사용해 도넛의 반지름을 조절할 수 있습니다.\
Minor Radius는 도넛 내부 반지름을 Major Radius는 도넛의 외부 반지름을 조절합니다.

### 2. 도넛 회전하기
기본적으로 방향키를 사용해서 도넛을 회전시킬 수 있습니다.\
